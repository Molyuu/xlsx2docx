# xlsx2docx
> [For English version,click me](https://translate.google.com)

将给定的Excel表格中的数据填入Word模板，使用 Python 实现

## 用户指南：
- 如何使用？
    1. 从 [Release 页面](https://github.com/Molyuu/xlsx2docx/releases) 获取最新的预编译可执行文件.
    2. 参考压缩包内的示例文件修改自己的 Excel 和 Word 模板
    3. 直接运行程序
- 已实现的功能
    1. 基本的转换
    2. 货币类型的数据的转换
